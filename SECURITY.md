# Política de segurança

Não publique senhas, cookies de sessão, tokens, arquivos de perfil ou capturas contendo dados pessoais em issues públicas.

Para relatar uma vulnerabilidade, use a opção **Report a vulnerability** na aba Security do repositório. Inclua a versão afetada, o impacto observado e passos mínimos para reprodução, sem anexar credenciais reais.

Os instaladores oficiais são disponibilizados somente em GitHub Releases deste repositório. Enquanto não houver assinatura Authenticode, valide também o SHA-256 publicado junto ao release. Hash comprova integridade do arquivo entregue; não substitui uma assinatura de código.

O Urus Launcher nunca precisa de um token do GitHub para consultar releases públicos. Tokens, certificados e chaves privadas não devem ser adicionados ao aplicativo nem ao repositório.
